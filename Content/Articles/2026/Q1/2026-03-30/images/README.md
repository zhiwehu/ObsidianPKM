## 配图使用建议

| 位置 | 配图 | 风格 |
|------|------|------|
| 封面 | 封面图 Prompt 1/2/3 | 视觉冲击，吸引点击 |
| 什么是 OpenClaw | 能力对比图 | 信息图，辅助理解 |
| 7大场景 | 场景展示图 | 直观展示应用范围 |
| 安装步骤 | 流程图 | 降低配置门槛感知 |
| 个人助理场景 | 使用场景图 | 情感共鸣 |
| 结尾 | 引导关注图 | 引导关注下期 |

**推荐封面风格：Prompt 1（插画风格）或 Prompt 3（真实场景感）**

建议生成 2-3 个版本，选择最符合账号风格的。
