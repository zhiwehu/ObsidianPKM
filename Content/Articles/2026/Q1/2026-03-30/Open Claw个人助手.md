## 什么是 OpenClaw

**OpenClaw**（中文名"龙虾"）是腾讯云推出的开源 AI 助手框架。

你可以把它理解为你的**私人 AI 总管家**——它本身不是 AI，而是组装和调度各种 AI 能力的平台。

打个比方：普通 AI 应用像一个只会一招的工人，OpenClaw 像一个能调配多个工人的办公室。你告诉它要做什么，它调用合适的 AI 能力来完成。

**它和手机上的 AI 应用有什么不同？**

| | 普通手机 AI 应用 | OpenClaw |
|---|---|---|
| **记忆能力** | 聊完就忘，下次重新开始 | 能记住你的习惯、偏好、规定 |
| **懂你的程度** | 千人一面 | 你来定义它的性格、角色、语气 |
| **能干的事** | 回答问题、写写文字 | 自动搜索信息、管理文件、发消息提醒 |
| **自动化** | 需要你每次手动操作 | 设置好之后自动运行 |

简单说：**普通 AI 是工具，OpenClaw 更像是你的数字分身。**

## 对普通人的用途

入门阶段，记住两个核心场景就够了。

### 个人助理：帮你处理日常事务

- 帮你规划日程，到点提醒你
- 帮你记住账号密码、重要日期、约定事项
- 帮你起草邮件、消息、报告
- 帮你做信息搜集和整理

### 数据管家：帮你管理文件和信息

- 发给它的文件，它会记住，之后问它要，它能准确找到
- 帮你整理文件夹、归纳资料
- 帮你记录读书笔记、会议纪要，随时查阅

不要一开始就想搞全套自动化。先把这两个场景用起来，感受它"记得住你"的能力，再逐步扩展。

## 快速安装配置（小白版）

> 高级配置（多 Agent 协作、自定义插件等）不在本文范围，有需要可以进付费群专门讲。

### 第一步：部署服务

OpenClaw 需要运行在服务器上。腾讯云有现成的一键部署模板，对着页面点几下就好，不需要敲代码。

步骤：
1. 登录 [腾讯云轻量应用服务器控制台](https://console.cloud.tencent.com/lighthouse)
2. 新建实例，选择应用模板
3. 搜索"OpenClaw"，选择官方镜像
4. 完成部署

### 第二步：接入大模型（关键步骤）

OpenClaw 装好后还没有"大脑"，需要接入大模型才能回答问题。

推荐使用**腾讯混元大模型**，接入简单，有免费额度。

步骤：
1. 在 [腾讯云控制台](https://console.cloud.tencent.com) 注册账号
2. 找到「混元大模型」或「API Key 管理」
3. 创建 API Key，复制保存
4. 回到 OpenClaw 后台，在「模型配置」中选择「混元」
5. 填入 API Key，保存

**也可以用其他模型：**
- 通义千问（阿里云百炼平台注册）
- DeepSeek（免费额度充足）
- GPT-4 / Claude（有境外账号的话）

### 第三步：设置角色和基本信息

告诉它你是谁，你需要什么。发一段话给它：

```
【我的基本信息】
姓名：[名字]
职业：[职业]
所在地：[城市]

【我的偏好】
- 回答简洁明了，不要废话
- 重要事项用【重要】标注
- 如果有多个方案，列出来让我选

【我希望你帮我做的事】
1. 帮我记住重要的事情和文件
2. 在我忘记的时候提醒我
3. 帮我搜集和整理信息
4. 辅助我写作和起草文字

【我的习惯要求】
- 每天晚上9点提醒我做10分钟冥想
- 如果我连续3天没跟进某事，温
```

它会记住这些，之后每次对话都会基于你的背景来回应。

### 第四步：接入微信（强烈推荐）

这是让 OpenClaw 真正变成"随身助手"的关键一步。

步骤：
1. 在 OpenClaw 后台找到「通道管理」
2. 选择「微信」，点击授权
3. 页面出现二维码，用手机微信扫一扫
4. 添加「ClawBot」为好友

绑好后，用微信发消息就能和 AI 对话，不用开电脑。

> ⚠️ **常见问题：** 接好微信后发现 AI 不回答，一直显示"正在思考"。通常是模型 API Key 没配置好，或者额度用完了。先检查模型配置和 Key 是否正确。

## 实例讲解

以下 7 个场景全部经过我自己实测，每个都有具体的操作步骤。

### 1. 如何配置安全保障

在让 AI 处理重要事务之前，先划好安全边界。

在角色配置中加入：
```
【安全规则】
- 不帮我完成任何涉及金钱转账的操作
- 不把我的个人信息发给第三方
- 涉及法律、医疗、金融的重要决策，先告诉我再执行
- 如果我的指令可能违反法律或道德，请拒绝并说明原因
```

先划红线，后面用起来才放心。

### 2. 如何设置角色、性格和主人信息

让 AI 更懂你，需要你花 5 分钟做初始化：

```
【我的性格偏好】
- 喜欢简洁直接的表达
- 重要的事情用 emoji 标注
- 希望给建议时同时给出理由

【我希望扮演的角色】
我的私人助理，主要帮我：
1. 整理信息、提醒事项
2. 起草文案和消息
3. 帮我记录和回忆重要资料
```

设置好之后，AI 的回答风格和内容都会更贴合你的需求。

### 3. 如何让它记住规定

告诉它你的日常规矩，让它帮你监督：

**健康习惯监督：**
```
请记住以下我的习惯要求：
- 每天喝水不少于8杯
- 每工作50分钟休息10分钟
- 晚上11点前放下手机

当我忘记时，提醒我。
```

**工作习惯监督：**
```
我的工作习惯：
- 每天早上9点前发送今日计划
- 收到重要消息后24小时内回复
- 每周五下午做一次工作回顾
```

设置好后，AI 会在对应时间提醒你，或在你偏离习惯时温和提示。

### 4. 如何让它能搜索网络信息

在 OpenClaw 后台安装「网页搜索」相关插件：
1. 找到「插件/Skills」商店
2. 搜索并安装搜索类插件
3. 测试：问它"今天有什么 AI 行业新闻？"

之后可以这样用：
- "帮我查一下 XX 产品的最新动态"
- "帮我搜集关于 [某个话题] 的资料"
- "最近一周 [某个领域] 有什么值得关注的"

### 5. 如何做信息采集并定时发送

**每日资讯推送（自动运行）：**
```
请每天早上8:30，帮我：
1. 搜索过去24小时 AI 行业最重要的3条新闻
2. 和我关注的方向相关的最新动态
3. 整理成300字以内摘要，通过微信发给我
```

设置一次，之后每天自动推送，不用手动去刷新闻。

**竞品监控：**
```
每周末搜集 [XX产品] 和 [XX产品] 的最新动态，
整理成简报，包含产品名、动态内容、相关链接。
```

### 6. 如何做日程提醒

让 OpenClaw 成为你的贴心秘书：

**设置提醒：**
```
- 明天上午10点提醒我开会，记得发会议资料
- 下周三提醒我交报告
- 每天晚上9点提醒我做10分钟冥想
```

**智能日程管理：**
```
当我告诉你我要做某事时，自动帮我：
1. 记录到日程表
2. 提前1天提醒我
3. 提前1小时再次提醒
```

**提前规划：**
每天晚上问它：
```
明天是 [日期]，我计划做 [简单描述计划]，
请帮我安排时间块，标注每个时间段的核心任务。
```

### 7. 如何做数据文件管理：发给它，后期再问它要

这是最有"AI 管家"感的场景，也是我个人最常用的功能。

**发给它：**
```
请帮我存档：[粘贴会议记录/资料内容]
标注：这是 [XX项目] 的需求文档
时间：[日期]
```

**过几天问它：**
```
上周三那个项目会议，主要讨论了什么决策？
我让你存档的那个 XX 资料，核心要点是什么？
帮我把过去一周存档的资料整理一下。
```

**我的使用习惯：**

发文件时加固定格式，让它的记忆带标签：
```
发送内容：[粘贴内容]
类型：会议纪要 / 产品需求 / 读书笔记 / 其他
主题：[一句话描述]
日期：[日期]
```

OpenClaw 的价值，归根结底就是三件事：**它记得住你、懂你的偏好、能自动帮你干事。**

花 30 分钟配置好，你会发现：原来 AI 可以这么贴心。
