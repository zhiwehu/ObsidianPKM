## 什么是 OpenClaw？

OpenClaw（中文名叫"龙虾"）是腾讯云推出的一款云端 AI 助手框架。

你可以把它理解为一个**会很多技能的 AI 总管家**：
- 接入了各种大模型（GPT、Claude、通义千问……）
- 可以安装各种技能插件（Skills）
- 支持接入多个社交平台

之前你可能需要在网页上使用它，现在**直接在微信里就能用**。

## 接入步骤（图文详解）

![微信接入步骤流程图](images/03-flowchart-wechat-access-steps.png)

### 第一步：进入应用管理

1. 登录 [腾讯云轻量应用服务器控制台](https://console.cloud.tencent.com/lighthouse)
2. 找到你想要接入微信的 OpenClaw 实例
3. 点击实例卡片上的**「配置」**按钮
4. 进入「应用管理」页面

### 第二步：选择微信通道

在 OpenClaw 配置面板中，找到**通道卡片**，选择「微信」，然后点击**「前往授权」**。

### 第三步：扫码接入

稍等片刻，页面上会出现一个**二维码**。

打开手机微信，用微信内置的**扫一扫**功能扫描这个二维码。

### 第四步：完成授权

扫码后，微信会提示你添加「ClawBot」为好友。

点击**「添加」**，就直接进入和它的对话页面了！

**恭喜！这时候你已经可以在微信里直接和 AI 对话了。**

---

## ⚠️ 重要：配置模型 API Key

刚添加好的 ClawBot 可能还不会回答你的问题——因为它还需要一个**大脑**（大模型）来驱动。

你需要为 OpenClaw 配置一个**模型 API Key**。

配置方法：

1. 在腾讯云控制台的「应用管理」页面
2. 找到「模型配置」选项
3. 填入你的 API Key（支持 GPT、Claude、通义千问、DeepSeek 等多种模型）

> 📌 具体配置步骤可以参考腾讯云官方的 [OpenClaw 一键部署指南](https://cloud.tencent.com/developer/article/2624003)。

配置完成后，你的微信 AI 助手就正式「活」过来了！

## 常见问题

**Q: 扫码没反应怎么办？**
A: 检查微信版本是否达到要求，或者刷新页面重新获取二维码。

**Q: 添加后没有反应？**
A: 检查是否已经配置了模型 API Key，这是必须的。

**Q: 支持企业微信吗？**
A: 目前主要支持个人微信，企业微信的支持可以关注官方更新。
