## 1. 项目概述

### 一句话定位

> **AITi — 面向所有层次用户的 AI 能力发现与学习平台**

帮助用户快速找到适合自己的 AI 工具（Agents、Plugins、平台、工作流），并通过精选内容学习如何使用 AI 提升生产力。

### 解决的问题

- **信息过载**：AI 工具爆发式增长，用户难以找到真正有价值的工具
- **入门门槛高**：Agent、自动化工作流等概念对普通用户不够友好
- **缺乏系统性**：没有一站式平台同时覆盖工具导航 + 使用指南

---

## 2. 目标用户

| 用户层次 | 特征 | 痛点 | AITi 能提供 |
|---------|------|------|------------|
| **初学者** | 刚接触 AI，不确定从哪里开始 | 不知道 AI 能做什么，害怕选错 | 简洁的分类导航，清晰的场景推荐 |
| **进阶用户** | 已用过 ChatGPT，想深入 Agent 领域 | 不知道有哪些 Agent 平台，如何组合使用 | Agent 平台对比、工作流案例 |
| **专业用户** | 开发者或 AI 深度用户 | 缺少系统化的工具对比和最佳实践 | 深度评测、技术集成指南、自动化编排 |

---

## 3. 收录范围

### 核心分类

| 分类 | 定义 | 示例 |
|------|------|------|
| **AI Agents** | 可自主执行任务的 AI 系统 | Claude Code, OpenClaw, CodeX, OpenCode, OpenManus, Manus, 扣子 Coze, 腾讯元器 |
| **Plugins & Skills** | 扩展 AI 能力的插件/技能包 | Claude Plugins, Claude Code Skills, VS Code AI 扩展 |
| **AI Platforms** | 可构建 Agent 的平台 | n8n, Dify, Coze, 腾讯元器, FastGPT, Flowise |
| **AI IDEs** | 内置 AI 能力的开发环境 | Cursor, Windsurf, Trae, GitHub Copilot, Codeium |
| **Prompt & Methods** | AI 使用方法论 | Prompt Engineering, RAG, CoT, ToT, Agent 设计模式 |
| **Automation** | AI 驱动的自动化工具 | Zapier AI, Make AI, IFTTT AI, Browserbase |

### 暂不收录

- 通用大模型（ChatGPT, Claude 等已有专门平台的）
- 纯 SaaS 工具（非 AI 相关）
- 硬件产品

## 4. 商业模式

### 阶段 1：内容平台（MVP）

- **联盟营销**：工具推荐链接 → 佣金（CPS/CPA）
- **赞助内容**：工具厂商付费展示
- **订阅通讯**：AI 工具周报（$5/月）

### 阶段 2：社区

- 用户可提交工具 → 社区投票
- 付费专题 / 工作流模板市场

### 阶段 3：产品化

- AITi 自动化工作流模板商店
- 企业 AI 工具采购咨询

## 6. SEO 策略

### 关键词矩阵

- **核心词**：AI tools, AI agents, AI workflow automation
- **中文词**：AI 工具导航, Agent 平台推荐, AI 工作流自动化
- **长尾词**：[tool name] review, best AI [category] 2026, how to use [tool]

### 内容策略

- 每个工具独立页面：标题 + 描述 + 特性 + 优缺点 + 官方链接
- 分类页独立 URL：`/categories/ai-agents`
- 每周新增 2-3 个工具
- 定期发布"最佳 AI Agent 2026"等专题
