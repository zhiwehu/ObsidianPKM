# Human-Agent-Kanban Project

## 📋 Project Overview

**Project Name:** Human-Agent-Kanban (人+AI 看板平台)

**Type:** AI Product / SaaS

**Stage:** 调研阶段（SPEC 已完成 v1.2）

**Start Date:** 2026-03

**Target Launch:** 2026-Q3 MVP

---

## 🎯 Project Vision

创建一个让人类和 AI Agent 共同协作的看板平台，让人类和 AI 都能通过看板了解任务信息，并让 AI 自动工作。

**核心价值主张：**
- 人类与 AI Agent 在同一看板协作
- AI 自动追踪任务状态
- AI 主动汇报进展
- 减少人工协调成本

## 🔧 Technical Approach

**技术栈（参考 SPEC）：**
- 前端：React 18 + TypeScript + Zustand + @dnd-kit
- 后端：Fastify + Prisma + PostgreSQL + Redis
- AI：OpenClaw（HTTP API 触发，无需 Webhook）
- 部署：待定

**核心功能：**
1. 看板视图（列表/看板）
2. 任务创建与分配
3. AI Agent 集成
4. 自动状态更新
5. 进度追踪与汇报
